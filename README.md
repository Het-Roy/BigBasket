## 🚀 Features

- **Responsive Design**: Optimized for different screen sizes, ensuring a seamless shopping experience on desktops, tablets, and mobile devices.
- **Multiple Page Navigation**: Includes a comprehensive home page, category-specific pages (Exotic Fruits & Veg, Beverages/Tea), and a dedicated login interface.
- **Interactive UI**: Hover effects, smooth transitions, and high-quality imagery to mimic the premium feel of the original site.
- **Category Browsing**: Users can browse through various categories like Fruits, Vegetables, and Teas with ease.

## 🛠️ Technologies Used

- **HTML5**: Semantic tags were used to structure the content for better SEO and accessibility.
- **CSS3**: Advanced styling including Flexbox/Grid for layouts, animations, and responsive media queries.
- **Iconography**: Used modern icon sets to represent various navigation elements and shopping tools.

## 📦 How to Run Locally

To view this project locally on your machine:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Het-Roy/BigBasket.git
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd BigBasket/bigBasket
    ```
3.  **Open in Browser**:
    Simply open the `index.html` file in your preferred web browser (Chrome, Firefox, Edge, etc.).
